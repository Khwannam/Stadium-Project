## 📌 รายละเอียดโครงงาน

Stadium Project เป็นโปรแกรมสำหรับจองสนามกีฬาแห่งหนึ่ง พัฒนาในรูปแบบโปรแกรม GUI ด้วยภาษา Java โดยใช้ Java Swing ในการออกแบบหน้าจอผู้ใช้

ระบบรองรับการจองสนามกีฬา 3 ประเภท ได้แก่
- สนามฟุตบอล
- สนามแบดมินตัน
- สนามบาสเกตบอล

ผู้ใช้สามารถเลือกประเภทสนาม วันที่ ช่วงเวลา และจำนวนชั่วโมง จากนั้นระบบจะคำนวณค่าบริการอัตโนมัติ และแสดงผลสรุปการจอง

ระบบรองรับการชำระเงินออนไลน์แบบจำลอง เมื่อผู้ใช้กดชำระเงิน ระบบจะแสดงสถานะ **“ชำระเงินแล้ว”** และสร้าง QR Code สำหรับแสดงรหัสการจอง (Booking ID) เพื่อใช้ยืนยันก่อนเข้าใช้สนาม

ข้อมูลการจองจะถูกบันทึกลงไฟล์ CSV เพื่อเก็บเป็นประวัติการจอง

## 🎯 วัตถุประสงค์ของโครงงาน

1. เพื่อฝึกทักษะการเขียนโปรแกรมด้วยภาษา Java  
2. เพื่อเรียนรู้แนวคิดการเขียนโปรแกรมเชิงวัตถุ (OOP)  
3. เพื่อฝึกออกแบบหน้าจอโปรแกรมด้วย Java Swing  
4. เพื่อฝึกการอ่านและเขียนไฟล์ข้อมูล

## 🛠 เทคโนโลยีที่ใช้

- ภาษา Java  
- Java Swing (GUI)  
- การจัดเก็บข้อมูลด้วยไฟล์ CSV  
- ไลบรารีสำหรับสร้าง QR Code (เช่น ZXing)  

---

## 🚀 คุณลักษณะของระบบ (MVP)

- เป็นโปรแกรมจองสนามกีฬาแห่งหนึ่ง  
- รองรับ 3 ประเภทสนาม (ฟุตบอล / แบดมินตัน / บาสเกตบอล)  
- ผู้ใช้สามารถเลือก  
  - ประเภทสนาม  
  - วันที่ (กรอกข้อความ)  
  - ช่วงเวลา (เช้า / บ่าย / เย็น)  
  - จำนวนชั่วโมง  

- ระบบคำนวณราคาตามประเภทสนามและจำนวนชั่วโมง  
- แสดงราคาต่อชั่วโมงและราคารวมทั้งหมด  
- รองรับการชำระเงินออนไลน์แบบจำลอง  
- เมื่อชำระเงินสำเร็จ ระบบแสดงสถานะ **“ชำระเงินแล้ว”**  
- สร้าง QR Code แสดงรหัสการจอง  
- บันทึกข้อมูลการจองลงไฟล์ CSV

## 🏗 โครงสร้างคลาสเบื้องต้น

- `Main` → เริ่มต้นการทำงานของโปรแกรม  
- `Booking` → เก็บข้อมูลการจองและคำนวณราคา  
- `Payment` → จัดการสถานะการชำระเงิน
